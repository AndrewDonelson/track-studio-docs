## Overview

TrackStudio is an AI-powered music video generation platform that automatically creates high-quality music videos from audio stems and lyrics.

### Key Features

- **STEM-Based Processing**: Separate vocal and music tracks
- **AI Image Generation**: Zimage creates section backgrounds
- **LLM Prompt Generation**: AI-generated image prompts from lyrics
- **Automated Video Composition**: 6-layer FFmpeg video pipeline
- **YouTube Integration**: Automatic upload with AI-generated metadata
- **Queue System**: Background processing with auto-resume
